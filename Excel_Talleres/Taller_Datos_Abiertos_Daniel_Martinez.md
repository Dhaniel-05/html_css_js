# Proyecto Académico
# Datos Abiertos, Gestión Administrativa y Excel Aplicado

---

# Introducción

El presente proyecto tiene como finalidad introducir a los estudiantes en el uso de datos abiertos gubernamentales aplicados a procesos de gestión administrativa, análisis financiero y toma de decisiones empresariales mediante herramientas ofimáticas como Microsoft Excel.

Los estudiantes deberán explorar el Portal Nacional de Datos Abiertos de Colombia, comprender cómo funciona la publicación de información pública y posteriormente aplicar dichos datos en la creación y análisis administrativo de una PYME ubicada en el departamento del Valle del Cauca.

---

# Objetivos del Proyecto

## Objetivo General

Aplicar herramientas de Excel y análisis de datos abiertos gubernamentales para desarrollar un caso práctico de gestión administrativa empresarial.

---

# Objetivos Específicos

- Comprender el concepto de datos abiertos.
- Navegar correctamente el Portal Nacional de Datos Abiertos.
- Identificar conjuntos de datos públicos útiles para la gestión administrativa.
- Aplicar funciones de Excel en escenarios empresariales.
- Analizar información financiera mediante balances e indicadores.
- Elaborar visualizaciones y análisis gráficos.
- Interpretar datos reales para la toma de decisiones.

---
